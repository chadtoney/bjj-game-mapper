### What We're Deploying

**BJJ Game Mapper** - A React-based Brazilian Jiu-Jitsu mind mapping application

### Where We're Deploying

**Azure Static Web Apps** - Microsoft's hosting platform for static web apps

### Key Facts

- **Cost**: $0/month (Free Tier)
- **Users Supported**: 5-10 (scalable to 100+ on Free Tier)
